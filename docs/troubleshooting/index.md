# Troubleshooting

This section records common problems I encounter in experiments, software usage, and data analysis, together with possible causes and solutions.

## What will be included

- Description of the problem
- Possible causes
- Troubleshooting order
- What I actually tested
- What finally worked
- Notes for future prevention

## Recording principle

I want to record problems as individual entries so that I can quickly search for them when the same issue appears again.

## Planned entries

- High background in western blot
- Low transfection efficiency
- Poor fluorescence signal
- Software installation issues
- Image processing errors
- Plot export problems
