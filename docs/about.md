# About

I am a graduate student using this website as a personal research knowledge base.

This site is built to organize what I learn in software tools, experimental protocols, data analysis workflows, and troubleshooting during my graduate training.

## Why I built this site

Instead of keeping scattered notes in different places, I want to maintain a structured and searchable website that can help me review, update, and reuse what I learn over time.

## Main focus

My current focus is to record:

- software that I learn to use in research
- laboratory methods and protocols
- data analysis workflows
- practical troubleshooting notes
