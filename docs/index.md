# Welcome

This website is my graduate research knowledge base.

I use it to organize what I learn in software tools, experimental protocols, data analysis workflows, and troubleshooting during my research training.

## What you can find here

### Software
Notes on software I use in research, including setup, common functions, practical workflows, and usage tips.

### Experimental Protocols
Step-by-step records of laboratory procedures, key parameters, critical details, and my own working versions.

### Data Analysis
Records of analysis workflows, image processing steps, plotting methods, and useful scripts or settings.

### Troubleshooting
Common problems, possible causes, troubleshooting logic, and solutions that worked in practice.

## Purpose of this site

This site is designed as a long-term personal knowledge base.

Instead of recording only what I did on a certain day, I want to organize knowledge by topic, so that it can be searched, reused, and updated over time.
